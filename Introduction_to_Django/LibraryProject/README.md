# LibraryProject is a basic Django application that serves as a foundation for learning Django development. 
It includes the default Django project structure with settings, URLs, and manage.py
## Setup

1. Create a virtual environment: `python3 -m venv venv`
2. Activate the environment: `source venv/bin/activate`
3. Install dependencies: `pip install django`
4. Run the server: `python3 manage.py runserver`
